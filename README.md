Python Management Suite

A collection of beginner-friendly Python applications demonstrating core programming fundamentals through multiple real-world management systems.


📌 Project Overview

This repository includes several mini systems built using Python. Each module focuses on solving a specific problem and demonstrates the use of basic programming concepts such as functions, conditionals, and data structures.


🚀 Features

- 📚 Student Record System  
- 📝 Student Marks Management  
- 🎓 University Course Management  
- 🎟️ Event Registration System  
- 🛒 Online Store Product List  


🛠️ Technologies Used

- Python  
- Lists & Dictionaries  
- Functions  
- Conditional Statements
